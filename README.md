# Go User API

Простой REST API на Go для создания и получения пользователей.  
Используется встроенный `net/http` и `sync.Mutex` для хранения данных в памяти.

---

## Функциональность

- Создание пользователя (`POST /users`)
- Получение пользователя по ID (`GET /users/{id}`)
- Логирование всех HTTP-запросов с методом, путем, статусом и временем обработки

---

## Структура проекта

.  
├── main.go # основной файл сервера  
└── go.mod # модуль Go

## Запуск проекта

1. Клонируйте репозиторий:

```bash
git clone https://github.com/ТВОЙ_USERNAME/go-user-api.git
cd go-user-api
```
Запустите сервер:
```bash
go run main.go
```
Сервер будет доступен по адресу: http://localhost:8080.
